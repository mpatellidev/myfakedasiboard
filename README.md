# DaSIboard
Dashboard com informações do curso de SI da USP.

O projeto conta com as seguintes abas: Home, Calendário, Horários, Newsletter e Docentes.

## Home
Permite visualizar os próximos eventos e a última newsletter, além de informações básicas do curso.

## Calendário
Visualizar eventos (provas, entregas, eventos, apresentações e deadlines) em formato de agenda.

## Horários
Selecionar e visualizar os horários das aulas para diferentes turmas e acessar o jupiterweb.

## Newsletter
Atualizações periódicas com novidades, notícias, coberturas, etc.

## Docentes
Database com os docentes da EACH, nome, e-mail, salas (variáveis), lattes e site.

# Updates
A pasta date possui 3 arquivos: event.json; newsletter.json; schedule.json

Qualquer usuário pode entrar nesses arquivos, adicionar um novo dado conforme os moldes de cada um e submeter para atualização via pull request.

# Issues
Problemas encontrados, sugestões e falhas críticas podem ser submetidos a análise via criação de issues no repositório.

# Projects
A aba projects possui um kanban com uma união dos issues e todas as futuras ideias para o projeto.
